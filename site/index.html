HOW IT WORKS

원인을 단정하지 않고,
확인할 근거를 모읍니다.

01

평소에는 기록

Windows 서비스가 UI를 닫아도 핵심 센서를 CSV에 계속 기록합니다.

02

종료 순간은 보존

비정상 종료로 열린 채 남은 세션과 직전 구간을 다음 부팅까지 지킵니다.

03

재부팅 뒤 확인

같은 시간축에서 온도·전력·제한 신호와 Windows 이벤트를 함께 읽습니다.

WHAT IT RECORDS

전원 종료 직전의
하드웨어 문맥

CPU와 GPU

온도, 순간 전력, 사용률, 클럭과 thermal·power 제한 신호

메인보드

지원되는 시스템의 VRM·칩셋 온도, 팬 회전수와 12V·5V·3.3V

종료 기록

마지막 샘플, Windows 이벤트와 센서별 min·max·last 요약

로컬 데이터

사용자명·호스트명·일련번호 없이 PC의 ProgramData에만 저장
