HOW IT WORKS

원인을 단정하지 않고,
확인할 근거를 모읍니다.

01

평소에는 기록

Windows 서비스가 UI를 닫아도 핵심 센서를 CSV에 계속 기록합니다.

02

종료 순간은 보존

갑자기 꺼져 정상적으로 끝나지 못한 기록과 종료 직전 데이터를 다음 부팅까지 보존합니다.

03

재부팅 뒤 확인

온도·전력·제한 신호와 Windows 이벤트를 같은 시간대에 나란히 확인합니다.

AI-ASSISTED REVIEW

기록을 복사하면,
AI가 결론부터 정리합니다.

종료 기록을 고르고 AI 진단 프롬프트 복사를 누르세요. PC 사양, Windows 이벤트와 마지막 센서값을 한 번에 붙여넣을 수 있는 요청문으로 정리합니다.

  1. 1

    종료 기록 선택확인할 시점을 고릅니다.

  2. 2

    프롬프트 복사진단에 필요한 내용만 모읍니다.

  3. 3

    AI에 붙여넣기결론·근거·다음 확인 순서로 답을 받습니다.

앱이 AI를 직접 실행하거나 데이터를 자동으로 보내지는 않습니다. 복사한 내용은 사용자가 확인한 뒤 원하는 AI 서비스에 붙여넣습니다.
